If y = -2 x 3^(x - 4) and y = 4^(-2x - 4) + 3 both come from exponential parent functions, how do we read the transformation story correctly and use it to predict domain, range, asymptote, and real-world growth?

Q1. For y = -2 × 3^(x - 4), what does the value a = -2 tell us?

err-a-vertical-transform | understand | L1

  1. ✓ A. Vertical stretch by factor 2 and reflection in the x-axis.
    Correct! a = -2 means vertical stretch by factor 2 (since |a|=2 > 1) and reflection in the x-axis (since a is negative).
  2. B. Vertical compression by factor 2 and reflection in the y-axis.
    Not quite. The negative sign causes a reflection in the x-axis, not the y-axis. Also, |a|=2 > 1 means a stretch, not compression.
  3. C. Horizontal stretch by factor 2 and reflection in the x-axis.
    Nearly. The coefficient a affects vertical, not horizontal transformations. Check which axis a affects.
  4. D. No vertical transformation, only reflection.
    Remember that a = -2 has both a magnitude (|a|=2) and a sign. The magnitude causes a vertical stretch by 2.

Q2. For y = 4^(-2x - 4) + 3, which horizontal transformations are caused by the -2 inside the exponent?

err-k-horizontal-transform | analyze | L2

  1. ✓ A. Reflection in the y-axis and horizontal compression by factor 1/2.
    Correct! k = -2 (from exponent -2x - 4 = -2(x+2)) causes a reflection in the y-axis and a horizontal compression by 1/|k| = 1/2.
  2. B. Reflection in the x-axis and horizontal stretch by factor 2.
    Almost. The negative sign reflects in the y-axis, not the x-axis. Also, |k|=2 leads to compression, not stretch.
  3. C. Horizontal stretch by factor 2 and no reflection.
    Check the sign: k is negative, so there is a reflection. Also, the scale factor is the reciprocal: 1/|k| = 1/2 compression.
  4. D. Reflection in the y-axis and horizontal stretch by factor 2.
    The magnitude |k|=2 gives a compression by 1/2, not a stretch by 2. Remember: compression when |k|>1.

Q3. For y = 1.5 × 2^(-x) + 2, what are the horizontal asymptote and range?

err-asymptote-range | apply | L2

  1. ✓ A. Asymptote y = 2; range y > 2.
    Correct! The vertical shift c = 2 moves the asymptote to y = 2. Since a > 0 and base > 0, the graph is above the asymptote, so range y > 2.
  2. B. Asymptote y = 0; range y > 0.
    Don't forget the vertical shift! The +2 moves the asymptote up to y = 2.
  3. C. Asymptote y = 2; range y < 2.
    Check the direction: because a = 1.5 is positive, the graph lies above the asymptote, not below.
  4. D. Asymptote y = 0; range y < 0.
    Two errors: the asymptote should be y = 2 (due to vertical shift) and the range should be y > 2 (above the asymptote).

Q4. A population starts at 35,000 and grows by 8% per year. Which model matches the lesson?

err-growth-factor | apply | L2

  1. ✓ A. P(x) = 35000 × 1.08^x
    Correct! Growth factor = 1 + 0.08 = 1.08. The model is P(x) = initial × (1 + rate)^x.
  2. B. P(x) = 35000 × 0.08^x
    Careful! 0.08 is the rate, not the factor. The growth factor is 1 + 0.08 = 1.08.
  3. C. P(x) = 35000 × 0.92^x
    0.92 would be a decay factor (1 - 0.08). For growth of 8%, use 1 + 0.08 = 1.08.
  4. D. P(x) = 35000 × 1.8^x
    1.8 would be an 80% growth. Remember: 8% growth means add 0.08 to 1, giving 1.08, not 1.8.

Q5. Explain why solving 70000 = 35000 × 1.08^x answers the question 'when will the population double?'

    补救模块

    err-a-vertical-transform

    Student misreads coefficient a and cannot connect it to vertical stretch/compression or reflection in the x-axis. They may think a negative sign causes a horizontal reflection or ignore the stretch factor.

    Explain: In the transformed exponential form F(x) = a × b^(k(x - d)) + c, the coefficient 'a' controls all vertical changes to the parent function y = b^x. The absolute value |a| tells you the vertical stretch (if |a| > 1) or compression (if 0 < |a| < 1). The sign of a tells you whether there is a reflection: if a is positive, the graph stays above the horizontal asymptote; if a is negative, the graph is reflected across the x-axis and lies below the asymptote. Remember: 'a' affects the y-values, not the x-values.

    Example: Consider y = -2 × 3^(x - 4). Here a = -2. Since | -2 | = 2 > 1, there is a vertical stretch by a factor of 2. Since a is negative, there is also a reflection in the x-axis. The parent points (0,1) and (1,3) would be stretched vertically (y multiplied by 2) and then reflected: (0, -2) and (1, -6). Without the reflection, they would be (0,2) and (1,6).

    Anchor: Think 'a' as the 'amplitude' of exponential: it stretches/squishes the graph vertically and flips it if negative. 'a' outside the base = vertical changes.

    err-k-horizontal-transform

    Student misreads k and confuses horizontal compression/stretch with vertical changes or uses the wrong reciprocal factor.

    Explain: In the transformed exponential form F(x) = a × b^(k(x-d)) + c, the parameter k affects the graph horizontally. If |k| > 1, the graph compresses horizontally by factor 1/|k| (it becomes steeper). If 0 < |k| < 1, the graph stretches horizontally by factor 1/|k| (it becomes wider). If k is negative, the graph reflects across the y-axis. A common mistake is to think |k| > 1 causes a stretch, but it's the opposite: think of k as a 'speed' factor—larger magnitude compresses the x-values.

    Example: Consider y = 4^(-2x-4) + 3. Rewrite the exponent as -2(x+2), so k = -2. Since |k| = 2 > 1, there is a horizontal compression by factor 1/2. The negative sign means a reflection in the y-axis. So the graph is compressed horizontally and flipped left-right.

    Anchor: Inside the exponent, k works like a camera zoom: |k| > 1 zooms in (compresses horizontally), 0 < |k| < 1 zooms out (stretches). A negative k flips the view horizontally.

    err-d-horizontal-shift

    Students often think that a positive d in x - d means a shift to the left, or they fail to factor the exponent to isolate the shift, especially when the exponent includes a coefficient. They may also confuse horizontal shift with vertical shift.

    Explain: The horizontal shift in an exponential function is determined by the value d in the form F(x) = a × b^(k(x - d)) + c. The graph of the parent function y = b^x is shifted to the right by d units if d is positive, and to the left by |d| units if d is negative. To find d, rewrite the exponent as k(x - d). For example, in y = -2 × 3^(x - 4), the exponent is already in the form 1·(x - 4), so k = 1 and d = 4, meaning a shift right 4 units. In y = 4^(-2x - 4) + 3, factor the exponent: -2x - 4 = -2(x + 2) = -2(x - (-2)), so d = -2, meaning a shift left 2 units. Always look for the expression (x - d) inside the exponent after factoring out k.

    Example: Consider y = 1.5 × 2^(-x) + 2. The exponent is -x = -1(x - 0), so d = 0, no horizontal shift. In y = -2 × 3^(x - 4), the shift is right 4. In y = 5 × 2^(x+3), rewrite as 5 × 2^(x - (-3)), so shift left 3.

    Anchor: Remember: 'x minus d' means shift right when d is positive. The sign of d is opposite the sign in the parentheses: (x - positive) shifts right.

    err-asymptote-range

    The student forgets that the vertical shift parameter c moves the horizontal asymptote to y = c, and then incorrectly determines the range as if the asymptote were still y = 0 or confuses which side of the asymptote the graph lies on.

    Explain: In the transformed exponential form F(x) = a × b^(k(x - d)) + c, the horizontal asymptote is determined solely by the vertical shift c: it is always y = c. The graph approaches this line but never crosses it. The range depends on whether the graph lies above or below the asymptote. If a > 0 and b > 0 (the base), the graph is above the asymptote, so range is y > c. If a < 0 (reflection in x-axis), the graph is below the asymptote, so range is y < c. Always identify c first, then check the sign of a to decide the direction.

    Example: For y = 1.5 × 2^(-x) + 2, c = 2, so the horizontal asymptote is y = 2. Since a = 1.5 > 0, the graph lies above the asymptote, so the range is y > 2. The domain is all real numbers.

    Anchor: The asymptote is the floor or ceiling; the graph lives either above or below it. C sets the floor/ceiling height.

    err-growth-factor

    The student incorrectly uses the decimal growth rate (0.08) as the base of the exponential function instead of converting it to the growth factor b = 1 + r. This mistake means each year the population is multiplied by 0.08 (shrinking drastically) instead of by 1.08.

    Explain: In exponential models, the base b is the factor by which the quantity changes each period. If a quantity increases by r% each period, the new value is (100% + r%) = (1 + r) times the previous value. So b = 1 + r. For example, 8% growth means b = 1 + 0.08 = 1.08. Similarly, for decay, subtract the rate: b = 1 - r.

    Example: A city's population grows by 8% per year. If the initial population is 35,000, the correct model is P(x) = 35000 × 1.08^x, not 35000 × 0.08^x. Because after one year, population = 35000 × 1.08 = 37,800, which is an 8% increase. Using 0.08 gives 35000 × 0.08 = 2,800, which is wrong.

    Anchor: Think '1 + rate' to create your base. For growth: b = 1 + r. For decay: b = 1 - r.
