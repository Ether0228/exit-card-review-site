Q1 / KN1 / E1 / Identify the domain of a polynomial function.

What is the domain of f(x) = x^3 - 6x^2 + 9x?

  1. A. All real numbersCorrect! Polynomial functions are defined for all real numbers.
  2. B. x ≠ 0Incorrect. There is no denominator or even root, so x=0 is allowed.
  3. C. x ≥ 0Incorrect. The function is defined for negative x as well.
  4. D. x > 3Incorrect. The function is defined for all x, not just x > 3.
Q2 / KN2 / E2 / Find the x-intercepts of a factored polynomial.

Given f(x) = x(x-3)^2, what are the x-intercepts?

  1. A. 0 and 3Correct! Setting each factor to zero gives x=0 and x=3.
  2. B. 0 onlyIncorrect. (x-3)^2=0 also gives x=3.
  3. C. 3 onlyIncorrect. The factor x gives x=0 as well.
  4. D. 0, 3, and -3Incorrect. (x-3)^2=0 gives only x=3, not -3.
Q3 / KN3 / E3 / Determine the oblique asymptote of a rational function.

For f(x) = (x^2)/(x-1), what is the equation of the oblique asymptote?

  1. A. y = x + 1Correct! Long division gives x + 1 + 1/(x-1), so the oblique asymptote is y = x + 1.
  2. B. y = x - 1Incorrect. The quotient from long division is x + 1, not x - 1.
  3. C. y = xIncorrect. The remainder term affects the quotient; it's x + 1.
  4. D. y = 1Incorrect. The oblique asymptote is a slanted line, not horizontal.
Q4 / KN4 / E4 / Interpret a sign chart for f' to determine intervals of increase.

Refer to the sign chart for f'(x) in Figure 1. On which interval(s) is f increasing?

  1. A. (-∞, 1) and (3, ∞)Correct! f' > 0 on (-∞,1) and (3,∞), so f is increasing there.
  2. B. (1, 3) onlyIncorrect. On (1,3), f' < 0, so f is decreasing.
  3. C. (-∞, 1) onlyIncorrect. f is also increasing on (3,∞).
  4. D. (3, ∞) onlyIncorrect. f is also increasing on (-∞,1).
Q5 / KN5 / E5 / Interpret a sign chart for f'' to determine concavity.

Refer to the sign chart for f''(x) in Figure 2. Where is f concave down?

  1. A. (-∞, 2)Correct! f'' < 0 on (-∞,2), so f is concave down there.
  2. B. (2, ∞)Incorrect. On (2,∞), f'' > 0, so f is concave up.
  3. C. (-∞, 1) and (3, ∞)Incorrect. Those intervals relate to f', not f''.
  4. D. (1, 3)Incorrect. That interval is from the f' sign chart, not f''.
Q6 / KN6 / E5 / Identify the inflection point from f'' sign change.

For f(x) = x^3 - 6x^2 + 9x, f''(x) = 6x - 12. Where is the inflection point?

  1. A. x = 2Correct! f''(2)=0 and concavity changes from negative to positive at x=2.
  2. B. x = 1Incorrect. x=1 is a critical point, not an inflection point.
  3. C. x = 3Incorrect. x=3 is a critical point, not an inflection point.
  4. D. x = 0Incorrect. f''(0) = -12 ≠ 0.
Q7 / KN7 / E6 / Choose the correct sketch given derivative information.

A function has f'(x) > 0 on (-∞,1), f'(x) < 0 on (1,3), f'(x) > 0 on (3,∞), and f''(x) < 0 on (-∞,2), f''(x) > 0 on (2,∞). Which graph could represent f?

  1. A. Graph with local max at x=1, local min at x=3, inflection at x=2Correct! f' changes + to - at x=1 (max), - to + at x=3 (min), and f'' changes sign at x=2 (inflection).
  2. B. Graph with local min at x=1, local max at x=3, inflection at x=2Incorrect. f' goes from + to - at x=1, so it's a max, not a min.
  3. C. Graph with local max at x=2, local min at x=1 and 3Incorrect. Critical points are at x=1 and 3, not 2.
  4. D. Graph with no extrema but inflection at x=2Incorrect. f' changes sign at x=1 and 3, so there are extrema.

Q9 / KN4 / E4 / Explain how to find critical points and their role.

Explain how to find the critical points of a function and what they tell you about the graph.

Rubric: Correct: Set f'(x)=0 and solve for x; also include where f'(x) undefined. Critical points are candidates for local extrema. Partial: mentions only f'=0 or only extrema. Incorrect: confuses with inflection points or intercepts.

Sample strong answer

To find critical points, compute f'(x), then solve f'(x)=0 and also find where f'(x) is undefined. These x-values are critical points. They tell us where the function might have local maxima or minima because the slope changes sign at these points.

Q10 / KN7 / E6 / Describe how to sketch a rational function given asymptotes and sign charts.

You are given the function f(x) = x^2/(x-1) with vertical asymptote x=1, oblique asymptote y=x+1, and sign charts showing f'(x) > 0 on (-∞,0) and (2,∞), f'(x) < 0 on (0,1) and (1,2), and f''(x) > 0 on (-∞,1) and (1,∞). Describe how you would sketch the graph near x=1 and at x=0 and x=2.

Rubric: Strong answer identifies the vertical asymptote x=1 and oblique asymptote y=x+1; describes f(x) -> -∞ as x -> 1- and f(x) -> +∞ as x -> 1+; identifies a local maximum at x=0 and a local minimum at x=2 from the f' sign changes; notes the graph is concave up on both sides of the vertical asymptote.

Sample strong answer

I would draw the vertical asymptote x=1 and the oblique asymptote y=x+1 first. As x approaches 1 from the left, f(x) goes to -∞; as x approaches 1 from the right, f(x) goes to +∞. Since f' changes from positive to negative at x=0, there is a local maximum at x=0. Since f' changes from negative to positive at x=2, there is a local minimum at x=2. The graph is concave up on both sides of the asymptote, so each branch should bend upward while approaching the asymptotes correctly.

补救模块

E1

Student does not recognize that polynomial functions have domain all real numbers.

Mini lesson: Polynomial functions are defined for all real numbers because you can plug any x into a polynomial and get a real output. There are no denominators, square roots of negatives, or logs of non-positives.

E2

Student incorrectly finds x-intercepts, possibly forgetting to set f(x)=0 or solving incorrectly.

Mini lesson: To find x-intercepts, set f(x) = 0 and solve for x. For factored forms, set each factor equal to zero. The y-intercept is found by evaluating f(0).

E3

Student confuses asymptote types or makes errors in long division for oblique asymptotes.

Mini lesson: Vertical asymptotes occur where denominator = 0 (after simplifying). Horizontal asymptotes: if degree numerator < degree denominator, y=0; if equal, y = leading coeff ratio. Oblique asymptotes: if degree numerator = degree denominator + 1, use long division; the quotient (without remainder) is the oblique asymptote.

E4

Student misinterprets sign chart of f' or incorrectly finds critical points.

Mini lesson: Critical points occur where f'(x)=0 or f'(x) is undefined. On a sign chart, f'(x) > 0 means f is increasing; f'(x) < 0 means f is decreasing. Local max where f' changes from + to -; local min where f' changes from - to +.

E5

Student misinterprets sign chart of f'' or incorrectly finds inflection points.

Mini lesson: Inflection points occur where f''(x)=0 or undefined AND concavity changes. f''(x) > 0 means concave up; f''(x) < 0 means concave down.

E6

Student cannot synthesize derivative information into a correct sketch, missing features or drawing inconsistent slopes/concavity.

Mini lesson: To sketch, plot intercepts, asymptotes, critical points, inflection points. Use sign charts to determine increasing/decreasing and concavity on intervals. Connect points smoothly respecting all behaviors.

E7

Student confuses the relationships between f, f', f'' graphs, e.g., thinking f' positive means f is concave up.

Mini lesson: f' gives slope of f: where f' > 0, f increases; where f' < 0, f decreases. f'' gives concavity of f: where f'' > 0, f is concave up; where f'' < 0, f is concave down. Also, f' extrema correspond to f'' zeros.
