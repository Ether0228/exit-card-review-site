补救模块

E01

Thinks that any quantity with a number is a vector, or that direction is optional.

Mini lesson: A scalar has only magnitude (size). Examples: mass, temperature, speed. A vector has both magnitude and direction. Examples: displacement, velocity, force. To decide, ask: Does it make sense to ask 'which way?' If yes, it's a vector.

E02

Thinks vectors are equal if they have the same length, regardless of direction.

Mini lesson: Equal vectors must have the same magnitude AND the same direction. Two arrows of the same length but pointing different ways are not equal. They can be in different locations, but the direction must be identical.

E03

Thinks opposite vectors are any vectors pointing in different directions, or that parallel vectors are always in the same direction.

Mini lesson: Opposite vectors have the same magnitude but exactly opposite directions (180° apart). Parallel vectors have the same or opposite direction (they are along the same line), but they can have different magnitudes. All opposite vectors are parallel, but not all parallel vectors are opposite.

E04

Thinks bearing is measured from east or counterclockwise.

Mini lesson: True bearing is measured clockwise from north, from 0° to 360°. North = 0°, east = 90°, south = 180°, west = 270°. For directions like southeast, it's halfway between south and east: 135°.
