Q1 / KN1 / E01 / recall

Which of the following quantities is a vector?

  1. A. Temperature of 25°CTemperature has only magnitude, so it is a scalar.
  2. B. Mass of 10 kgMass has only magnitude, so it is a scalar.
  3. C. Velocity of 20 m/s northCorrect! Velocity has both magnitude (20 m/s) and direction (north).
  4. D. Time of 5 secondsTime has only magnitude, so it is a scalar.
Q2 / KN2 / E04 / compute

A vector has components 3 units east and 4 units north. What is its magnitude?

  1. A. 7 unitsIncorrect. You added the components directly. Use the Pythagorean theorem: sqrt(3^2+4^2)=5.
  2. B. 5 unitsCorrect! sqrt(3^2+4^2)=sqrt(9+16)=sqrt(25)=5.
  3. C. 12 unitsIncorrect. You multiplied the components. Use the Pythagorean theorem: sqrt(3^2+4^2)=5.
  4. D. 25 unitsIncorrect. You forgot to take the square root. sqrt(3^2+4^2)=5, not 25.

Q5 / KN6 / E06 / judge

True or False: If two vectors are parallel, they must be equal.

  1. A. TrueIncorrect. Parallel vectors can have different magnitudes. For example, two vectors pointing east with lengths 3 and 5 are parallel but not equal.
  2. B. FalseCorrect! Parallel vectors have the same or opposite direction but may have different magnitudes.
  3. C. Only if they point the same directionIncorrect. Even if they point the same direction, they may have different magnitudes.
  4. D. Only if they have the same magnitudeIncorrect. They also need the same direction to be equal.

Q9 / KN2 / E04 / compute

A vector has components 6 units west and 8 units south. Calculate its magnitude. Show your work.

Rubric: Correct answer: 10 units. Work: sqrt(6^2+8^2)=sqrt(36+64)=sqrt(100)=10. Accept 10 units or 10.

Sample strong answer

Magnitude = sqrt(6^2 + 8^2) = sqrt(36 + 64) = sqrt(100) = 10 units.

Q10 / KN3 / E05 / construct

A ship sails 50 km on a true bearing of 120°. Describe its direction using compass quadrants (e.g., N30°E).

Rubric: Correct: S60°E (since 120° clockwise from north is 60° east of south). Accept equivalent descriptions like '60° east of south' or 'S60°E'.

Sample strong answer

The direction is S60°E, meaning 60° east of south.

补救模块

E01

Thinks a vector is defined only by its magnitude or confuses vector with scalar.

Mini lesson: A vector has both magnitude and direction. A scalar has only magnitude. For example, speed (scalar) vs velocity (vector).

E02

Confuses equal vectors with identical vectors (same position).

Mini lesson: Equal vectors have the same magnitude and direction, regardless of starting point. They are parallel and point the same way.

E03

Thinks parallel means same direction only.

Mini lesson: Parallel vectors can be in the same or opposite direction. Opposite vectors are always parallel because they lie on parallel lines.

E04

Misapplies Pythagorean theorem.

Mini lesson: Magnitude = sqrt(x^2 + y^2). Do not add components directly; square each, sum, then take square root.

E05

Thinks bearing is measured from east or counterclockwise.

Mini lesson: True bearing is measured clockwise from north, always three digits (e.g., 045°). Quadrant bearing uses N/S and E/W (e.g., N45°E).

E06

Ignores magnitude requirement for equality.

Mini lesson: Parallel vectors have same or opposite direction, but may have different magnitudes. Equal vectors must have same magnitude AND direction.

E07

Confuses which end is tail and which is head.

Mini lesson: A vector is drawn from tail to head. The arrowhead points to the head. The direction is from tail to head.
