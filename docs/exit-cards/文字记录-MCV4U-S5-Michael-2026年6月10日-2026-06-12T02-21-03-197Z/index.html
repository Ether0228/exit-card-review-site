Q1 / KN1 / E1 / Identify the domain of a polynomial function.

What is the domain of the function f(x) = 2x^3 - 5x^2 + 3x - 7?

  1. A. All real numbersCorrect! Polynomials are defined for all real numbers.
  2. B. x ≠ 0There is no division by x, so x=0 is allowed.
  3. C. x ≥ 0Polynomials are defined for negative x as well.
  4. D. x > 0Polynomials are defined for all real numbers, including zero and negatives.
Q2 / KN2 / E2 / Find the x-intercepts of a factored polynomial.

Given f(x) = x(x - 2)(x + 1), what are the x-intercepts?

  1. A. 0, 2, -1Correct! Setting each factor to zero gives x=0, x=2, x=-1.
  2. B. 0, -2, 1Check the signs when solving x-2=0 and x+1=0.
  3. C. 2, -1 onlyDon't forget the factor x, which gives x=0.
  4. D. 0 onlyThere are three factors, so three x-intercepts.

Q9 / KN7 / E7 / Synthesize derivative evidence into graph-feature predictions without drawing the final graph.

Given f'(x) > 0 for x < -1 and x > 2, f'(x) < 0 for -1 < x < 2, f''(x) < 0 for x < 0.5, and f''(x) > 0 for x > 0.5, list the increasing/decreasing intervals, concavity intervals, and the likely x-values of local extrema and inflection. Do not draw the final graph.

Rubric: Strong answer identifies: increasing on (-∞,-1) and (2,∞); decreasing on (-1,2); concave down on (-∞,0.5); concave up on (0.5,∞); local maximum near x=-1; local minimum near x=2; inflection candidate at x=0.5 because f'' changes sign.

Sample strong answer

f is increasing on (-∞,-1) and (2,∞), and decreasing on (-1,2). It is concave down on (-∞,0.5) and concave up on (0.5,∞). Since f' changes from positive to negative at x=-1, there is a local maximum there. Since f' changes from negative to positive at x=2, there is a local minimum there. Since f'' changes from negative to positive at x=0.5, x=0.5 is an inflection candidate.

Q10 / KN7 / E6 / Explain the relationship between f, f', and f'' in curve sketching.

Explain how the signs of f'(x) and f''(x) determine the shape of the graph of f(x). Use the terms increasing, decreasing, concave up, and concave down in your explanation.

Rubric: Must state: f'(x) > 0 implies f increasing, f'(x) < 0 implies f decreasing; f''(x) > 0 implies concave up, f''(x) < 0 implies concave down. May mention critical points and inflection points.

Sample strong answer

The sign of f'(x) tells us whether f is increasing or decreasing: if f'(x) > 0, the graph goes up as x increases; if f'(x) < 0, it goes down. The sign of f''(x) tells us about concavity: if f''(x) > 0, the graph is concave up (curving upward like a cup); if f''(x) < 0, it is concave down (curving downward like a cap). Together, they determine the overall shape, such as increasing and concave down, or decreasing and concave up.

补救模块

E1

Student does not recognize that polynomials have domain all real numbers, or incorrectly applies restrictions from rational/root functions to polynomials.

Mini lesson: The domain of a function is the set of all x-values for which the function is defined. Polynomial functions (sums of powers of x with real coefficients) are defined for all real numbers because you can plug any real number into a polynomial and get a real output. There are no division by zero or even roots of negative numbers to worry about.

E2

Student makes factoring errors or forgets to set y=0 to find x-intercepts, or sets x=0 incorrectly for y-intercept.

Mini lesson: To find x-intercepts, set y=0 and solve for x. Factor the polynomial if possible. Each factor gives an x-intercept. To find the y-intercept, set x=0 and evaluate f(0).

E3

Student confuses vertical asymptotes (denominator=0) with x-intercepts, or incorrectly determines horizontal/oblique asymptotes by not comparing degrees.

Mini lesson: For rational functions f(x)=p(x)/q(x): Vertical asymptotes where q(x)=0 (and p(x)≠0). Horizontal asymptote: if degree(p) < degree(q), y=0; if equal, y=leading coeff ratio; if degree(p) > degree(q), no horizontal asymptote (but may have oblique). Oblique asymptote if degree(p) = degree(q)+1; find by polynomial long division.

E4

Student makes mistakes in differentiation rules, algebra when solving f'(x)=0, or misinterprets the sign chart (e.g., thinking positive derivative means decreasing).

Mini lesson: First derivative f'(x) gives slope. Critical points where f'(x)=0 or undefined. Use a sign chart: test points in each interval between critical points. If f'(x)>0, f is increasing; if f'(x)<0, f is decreasing. A critical point is a local max if f' changes from + to -, local min if - to +.

E5

Student incorrectly computes second derivative, misinterprets sign (e.g., f''>0 means concave down), or fails to find inflection points where f'' changes sign.

Mini lesson: Second derivative f''(x) tells concavity. f''(x)>0 → concave up (cup shape); f''(x)<0 → concave down (cap shape). Inflection points occur where f''(x)=0 or undefined AND concavity changes (sign of f'' changes).

E6

Student can compute derivatives but cannot translate sign information into the actual shape of the graph (e.g., drawing increasing when derivative is negative).

Mini lesson: The graph of f is a visual representation of all the derivative information. f' tells slope: positive f' means graph goes up as x increases; negative f' means graph goes down. f'' tells bending: positive f'' means graph curves upward (like a cup); negative f'' means curves downward (like a cap). Combine them: e.g., f'>0 and f''<0 means increasing but bending downwards (getting less steep).

E7

Student may correctly find individual features but fails to combine them into a coherent graph, missing asymptotes, inconsistent behavior near asymptotes, or not labeling key points.

Mini lesson: Curve sketching is like assembling a puzzle. Start by drawing asymptotes as dashed lines. Plot intercepts and critical/inflection points. Then use sign charts to sketch the curve's direction and bending in each interval. Ensure the graph approaches asymptotes correctly (e.g., vertical asymptote: graph goes to ±∞; horizontal/oblique: graph approaches the line as x→±∞).
