Q9 / node-graph-relationship / err-derivative-graph-relationship / evaluate L3

Explain how you would decide which graph is f, f', and f'' if the labels were removed. Use slope and concavity evidence.

Rubric: A strong answer states that f' is positive where f increases and negative where f decreases, and that f'' is positive where f is concave up and negative where f is concave down. It should mention that x-axis crossings/sign changes of f' and f'' correspond to extrema and inflection evidence.

Sample strong answer

I would start with the graph I think is f and check its slope. Where f is increasing, f' should be above the x-axis; where f is decreasing, f' should be below. Then I would check concavity: where f bends upward, f'' should be positive, and where it bends downward, f'' should be negative.

Q10 / node-integrated-sketch / err-integrated-curve-sketch-summary / create L3

Write a curve-sketching summary for f(x)=x^3-6x^2+9x. Include domain/continuity, intercepts, local extrema, inflection point, and concavity.

Rubric: A strong answer includes all real domain and continuity, intercepts (0,0) and (3,0), local maximum (1,4), local minimum (3,0), inflection point (2,2), increasing/decreasing intervals, and concave down/up intervals.

Sample strong answer

The function is a polynomial, so it is continuous for all real x and has no asymptotes. Its x-intercepts are (0,0) and (3,0). From f', it increases on (-infinity,1), decreases on (1,3), then increases on (3,infinity), so (1,4) is a local maximum and (3,0) is a local minimum. From f'', it is concave down before x=2 and concave up after x=2, so (2,2) is an inflection point.

补救模块覆盖

err-domain-continuity-function-type

Treats all functions the same and misses that polynomial vs rational changes domain/asymptote work.

Mini lesson: Start curve sketching by classifying the function. A polynomial has domain all real numbers and is continuous everywhere; a rational function needs denominator restrictions.

err-intercepts-vs-asymptotes

Confuses x-intercepts, discontinuities, and asymptotes.

Mini lesson: Intercepts are where the graph crosses an axis. Asymptotes come from restrictions or end behavior. They answer different questions.

err-critical-points-from-first-derivative

Does not correctly solve f'(x)=0 or connect critical numbers to local extrema.

Mini lesson: Critical numbers come from f'(x)=0 or where f' is undefined. They are x-values to test for local extrema.

err-sign-chart-to-intervals

Can make or read a sign chart but cannot translate signs into increasing/decreasing intervals.

Mini lesson: A positive f' interval means f is increasing; a negative f' interval means f is decreasing. The sign chart must become interval language.

err-concavity-inflection-second-derivative

Uses f''=0 mechanically without checking concavity change or point coordinates.

Mini lesson: f'' controls concavity. A point of inflection requires a change in concavity, and the final point must be on f, not on f''.

err-rational-oblique-asymptote

Misses vertical restrictions or oblique asymptote from long division in rational curve sketching.

Mini lesson: For rational functions, set the denominator to zero for vertical asymptotes, then compare degrees. If the numerator degree is exactly one higher, use long division for an oblique asymptote.

err-derivative-graph-relationship

Cannot connect f's slope to f' and f's concavity to f''.

Mini lesson: Use graph behavior as evidence. f' is tied to slope of f; f'' is tied to concavity of f.

err-integrated-curve-sketch-summary

Knows isolated facts but cannot assemble a coherent final curve-sketching summary.

Mini lesson: A complete curve sketch is a chain: domain, intercepts/asymptotes, f' sign, f'' sign, then final graph behavior.
