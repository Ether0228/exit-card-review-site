Use the figure. Explain how the sign of f' and f'' relate to the shape of f. Provide an example using f(x)=x^3-6x^2+9x.

Uses the example: on (-∞,1), f' > 0 so f increasing; on (1,3), f' < 0 so f decreasing; on (3,∞), f' > 0 so f increasing. f'' < 0 on (-∞,2) so concave down; f'' > 0 on (2,∞) so concave up. Inflection at x=2.

Sample strong answer

For f(x)=x^3-6x^2+9x, f'(x)=3x^2-12x+9. f' > 0 on (-∞,1) and (3,∞), so f is increasing there. f' < 0 on (1,3), so f decreasing. f''(x)=6x-12. f'' < 0 on (-∞,2) so concave down; f'' > 0 on (2,∞) so concave up. Inflection point at x=2 because concavity changes.

Q10 / KN04 / E04 / apply L2

Use the figure. For f(x)=x^2/(x-1), identify the vertical and oblique asymptotes and explain how you find them.

Rubric: Full credit: Vertical asymptote at x=1 because denominator zero. Oblique asymptote y=x+1 because degree numerator = degree denominator +1; use long division: x^2/(x-1)=x+1+1/(x-1).

Sample strong answer

Vertical asymptote: set denominator x-1=0 → x=1. Oblique asymptote: since numerator degree (2) is one more than denominator degree (1), there is no horizontal asymptote. Perform long division: (x^2) ÷ (x-1) = x+1 with remainder 1. So oblique asymptote is y=x+1.

补救模块

E01

Student may think polynomials have restrictions or confuses intercepts with asymptotes.

Mini lesson: Polynomial functions are defined for all real numbers. To find x-intercepts, set f(x)=0 and solve. For rational functions, domain excludes values that make denominator zero.

E02

Student may think f' > 0 means decreasing or vice versa.

Mini lesson: If f'(x) > 0 on an interval, f is increasing. If f'(x) < 0, f is decreasing. Use a sign chart to determine intervals.

E03

Student may think f'' > 0 means concave down or vice versa.

Mini lesson: If f''(x) > 0, f is concave up (like a cup). If f''(x) < 0, f is concave down (like a frown).

E04

Student may confuse vertical asymptote with x-intercept or forget to check degree for oblique asymptote.

Mini lesson: Vertical asymptotes occur where denominator = 0 (and numerator ≠ 0). Oblique asymptotes occur when degree of numerator = degree of denominator + 1; find by long division.

E05

Student may think f' above x-axis means f concave up, or confuse inflection points on f with zeros of f''.

Mini lesson: f' above x-axis (positive) means f increasing. f' below x-axis (negative) means f decreasing. f'' positive means concave up; f'' negative means concave down. Inflection points on f occur where f'' changes sign (or f''=0).
